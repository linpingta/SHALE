# SHALE-Cplusplus
C++ version of SHALE paper, ad inventory allocation

## Paper
[SHALE: An Efficient Algorithm for Allocation of Guaranteed Display Advertising](https://arxiv.org/pdf/1203.3619.pdf)

## How to run code

1. do cmake
```
cmake .
```
2. make
```
make
```
3. run binary
```
./shale
```
